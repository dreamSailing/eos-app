# EOS App v0.1.0-beta.1

本版本是 EOS App（Wails v3 桌面客户端）的首个 beta 预发布版本，基于 Web 技术栈构建桌面 GUI，支持对话、规则管理、工作区等核心功能。

## 下载

- Windows x64 安装包（Inno Setup）
  `eos-app-setup-0.1.0-beta.1.exe`
- Windows x64 便携版
  `eos-app-v0.1.0-beta.1-windows-amd64.zip`
- 可执行文件（未压缩）
  `eos-app-v0.1.0-beta.1-windows-amd64.exe`

## 更新内容

### 新增

- 基于 Wails v3 的桌面 GUI 客户端，支持对话交互。
- 支持全局与工作区规则独立管理。
- 支持工作树管理与 MCP JSON 导入。
- 支持多语言和主题切换。
- 支持沙箱模式与访问权限控制。
- 新增 Windows 安装包（Inno Setup），支持开始菜单与桌面快捷方式。
